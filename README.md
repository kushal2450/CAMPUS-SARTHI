# CAMPUS-SARTHI
A comprehensive placement portal for students to browse companies, check eligibility, and track placement analytics.

## Features

- 🔐 User Authentication (Register/Login)
- 🏢 Company Listings with Eligibility Criteria
- ✅ Automatic Eligibility Check
- 📊 Placement Analytics Dashboard
- 🎯 Personalized Company Recommendations

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)

## Setup Instructions

### 1. Install Prerequisites

- **Node.js** (v18 or higher)
  - Download from: https://nodejs.org
  - Install with default settings

- **MongoDB** (v8 or higher)
  - Download from: https://www.mongodb.com/try/download/community
  - Choose: Version 8.x, Platform Windows, Package MSI
  - During installation, check "Install MongoDB as a Service"

### 2. Clone or Download the Project

```bash
git clone <your-repo-url>
